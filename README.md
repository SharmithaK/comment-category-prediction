# Comment Category Prediction using Machine Learning

## Overview
This project focuses on building a Machine Learning model to classify user-generated comments into different categories using Natural Language Processing (NLP) and structured feature engineering techniques.

The goal is to analyze comment data and predict the appropriate category using multiple machine learning approaches.

---

## Project Workflow

### 1️. Data Preprocessing
- Handled missing values
- Cleaned and standardized text data
- Converted categorical and boolean features into numerical format

---

### 2️. Exploratory Data Analysis (EDA)
- Analyzed class distribution
- Studied feature relationships and patterns
- Performed outlier and skewness analysis

---

### 3️. Feature Engineering
- TF-IDF (word and character level representation)
- Engineered features from text and metadata
- Created engagement-based features

---

### 4️. Model Building
The following models were explored:
- Logistic Regression  
- LinearSVC  
- SGD Classifier  
- LightGBM  
- XGBoost  

---

### 5️. Optimization Techniques
- Feature selection (SelectKBest)
- Hyperparameter tuning
- Class imbalance handling using class weights

---

### 6️. Final Pipeline
A full machine learning pipeline was built combining:
- Text features  
- Numerical features  
- Categorical features  

---

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, LightGBM, XGBoost, NLP (TF-IDF), Matplotlib, Seaborn

---

## Key Learnings
- End-to-end ML pipeline development  
- Handling real-world noisy data  
- Feature engineering for NLP + structured data  
- Model evaluation and tuning techniques
